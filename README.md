# Internet Journey

An interactive educational web experience that visualizes what happens behind the scenes when a user types a URL into their browser.

## Overview

**Internet Journey** breaks down the lifecycle of a web request into visually engaging, scroll-driven sections to explain how the internet and browser rendering pipeline work together to load a webpage.

The project is designed to make complex web concepts more intuitive through animation, responsive design, and interactive storytelling.

---

## Features

* **DNS Resolution Visualization**
  Demonstrates how browsers locate a website’s IP address through DNS lookup.

* **Server Communication Simulation**
  Shows the HTTP request/response cycle between browser and server.

* **Data Transfer Representation**
  Visualizes packets/assets being transferred to the browser.

* **Browser Rendering Pipeline**
  Explains parsing, DOM creation, CSS application, and UI painting.

* **Performance Timeline Finale**
  Summarizes the full request lifecycle in an animated timeline.

* **Responsive Design**
  Optimized for desktop and mobile screens.

* **Accessibility Support**
  Includes skip links, focus states, and reduced-motion handling.
